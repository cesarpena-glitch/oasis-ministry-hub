Parking Lot — 1–2 officers (traffic and perimeter patrol)
Children's Wing — 1 officer near O'Kids classrooms
Roving Officer — covers hallways and restrooms
Command Post — Team Lead monitors cameras and radio
Emergency Protocols
Code Red — Active threat: lockdown immediately, call 911, notify team lead
Code Blue — Medical emergency: call 911, retrieve AED, clear the area
Code Yellow — Suspicious person: observe, report, do not confront alone
Evacuation — Use designated routes, direct congregation to parking lot
All incidents must be logged via the Incident Report Form within 24 hours
